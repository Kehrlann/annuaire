Le formulaire de recherche à gauche de l'écran vous permet de rechercher des anciens dans la base de données Intermines, en combinant différents critères. Afin de rendre la recherche plus souple et efficace, les règles suivantes s'appliquent :

Recherche étendue :
Si vous recherchez un mot, le moteur recherche fera une recherche sur tous les groupes contenant ce mot. Par exemple, rechercher l'entreprise thalès permettra de trouver tous les anciens travaillant chez Thalès, mais aussi chez Thalès Avionics, Thalès Underwater Systems, etc.

Recherche exacte :
Si vous souhaitez effectuer une recherche sur un terme exact, veuillez placer le terme entre guillements. Par exemple, "edf" permet de rechercher l'entreprise EDF en ignorant les résultats comme EDF GDF, EDF RTE, etc.

Recherche multiple :
Vous pouvez rechercher sur plusieurs termes à la fois, en les séparant par un espace. Par exemple, faire une recherche sur l'entreprise sncf ratp trouvera les anciens travaillant pour la SNCF ainsi que ceux travaillant pour la RATP.

Recherche par promotion :
Vous pouvez rechercher sur une promotion unique (ex :2008), sur plusieurs promotions à la fois (ex :1990 1995) ou sur un ensemble de promotions (ex : 2000-2010).
